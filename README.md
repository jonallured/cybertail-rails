# README

[![Build Status][status]][icon]

## Setup

* Clone the project

* Read and run setup script:
  ```
  $ cat bin/setup
  $ bin/setup
  ```

## Tests

Once setup, you can run the tests like this:

```
$ bundle exec rake spec
```

Note: the default rake task is setup to run tests and Rubocop.
